# AGENTS.md

## Main Communication Language

- 이 프로젝트의 기본 커뮤니케이션 언어는 한국어입니다.
- 사용자가 다른 언어를 명시적으로 요청하지 않는 한, 답변과 작업 요약은 한국어로 작성합니다.
- 코드, 파일명, 명령어, 게임 내부 식별자, API 이름은 원문 표기를 유지합니다.

## Project Overview

이 프로젝트는 `Car Mechanic Simulator 2021` 게임에서 `Rubber Bushing` 부품이 절대 고장나지 않도록 만드는 모드입니다.

## Working Guidelines

- 기존 파일과 구조가 생기면 먼저 현재 구현 방식과 명명 규칙을 확인한 뒤 수정합니다.
- 모드의 목적은 좁고 명확합니다: `Rubber Bushing`의 마모, 손상, 고장, 교체 필요 상태를 방지합니다.
- 다른 부품의 내구도, 경제 시스템, 정비 플로우는 사용자가 요청하지 않는 한 변경하지 않습니다.
- 게임 파일, 설정 파일, 패치 로직을 다룰 때는 변경 범위를 최소화하고 되돌리기 쉬운 방식으로 작업합니다.
- 테스트 또는 검증 절차가 가능하다면, 변경 후 어떤 방식으로 확인했는지 한국어로 간단히 기록합니다.

## Notes for Future Agents

- 이 저장소의 목표는 범용 CMS 2021 모딩 프레임워크가 아니라 `Rubber Bushing` 고장 방지 모드입니다.
- `Rubber Bushing`과 관련 없는 밸런스 변경이나 광범위한 리팩터링은 피합니다.
- 사용자에게 보고할 때는 핵심 변경 사항, 검증 여부, 남은 확인 사항을 짧고 분명하게 전달합니다.
